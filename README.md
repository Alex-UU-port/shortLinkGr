В соответствии с заданием разработано консольное приложение. Для быстрой проверки в приложении имеется зарегистрированный пользователь login: Alice ; password: 11
У пользователя Alice имеется короткая ссылка. В приложении интуитивно понятная навигация, для перемещения по которому необходимо вводить цифры.

Приложение поддерживает:

- проект использует gradle;
- алгоритм сокращения и механизма уникальности ссылок для каждого пользователя;
- регистрацию и авторизацию пользователей;
- CLI-взаимодействие и цикл команд;
- сохранение всех пользователей и коротких ссылок в файлы "users.json" и "links.json";
- реализован переход/редирект по ссылке из консоли;
- реализован лимит переходов (подсчёт и блокировка);
- реализовано время жизни и автоудаление по сроку;
- реализованы UUID-идентификация и права на правку/удаление;
- реализована конфигурации приложения из отдельного конфигурационного файла "config.properties";
- реализован мультипользовательский режим и несколько ссылок у одного;
- имеются уведомления об исчерпании лимита/TTL;
- имеется возможность редактирования лимита и параметров (администрирование владельцем);
- имеется обработка ошибок и устойчивость;
- UX CLI: помощь / подсказки / понятные сообщения.
